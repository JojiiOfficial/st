## Install <br> 
`cd st & make && sudo make install `<br>

## Keybindings<br> 
<pre>
ctrl + shift + c        Copy  <br>
ctrl + shift + v        Paste <br>
alt  + comma            Zoom in <br>
alt  + .                Zoom out <br>
alt  + g                Reset Zoom<br>
alt  + s                Increase Transparency<br>
alt  + a                Decrease Transparency <br>
</pre>
you can change all of these in config.h
<br>
